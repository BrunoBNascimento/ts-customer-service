# Simple rest crud using mongoose and typescript
### Dependencies
 - [X] MongoDB
 - [X] ts-node
 - [X] Typescript
 - [X] NodeJS
 - [X] Yarn (or any npm client)

### How to start?

run command `yarn dev`

It will be increased with CQRS and Event Sourcing architeture
